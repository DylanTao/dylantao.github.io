Failure XP / Nerdy Spooder-Man Level 3 38 / 50 XP

Small joke metric for failed submissions. Combo and milestone badges add 0 XP so the same rejection is not counted twice.

  • First-author paper rejection +12 XP
  • Co-first listed first +11 XP
  • Co-first listed second +10 XP
  • Co-first listed third +9 XP
  • Co-author paper rejection +4 XP
  • Grant/proposal rejection +7 XP
  • Fellowship rejection +7 XP
  • Combo/milestone badges 0 XP They celebrate patterns without double counting.
  • Accepted work 0 XP here Accepted work already appears in the publication list.
