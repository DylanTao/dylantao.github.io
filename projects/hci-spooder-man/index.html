How to remix this

Hand the prompt and kit to your coding agent, then make it yours.

The move is simple: keep the serious portfolio trustworthy, attach one playful artifact, and publish the scaffold so the next person can adapt it.

  1. 01 Give your agent the prompt. Start with the markdown prompt below, then replace my research context with yours: field, site tone, colors, and the part of research you want to make visible.
  2. 02 Drop in the remix kit. Use the ZIP if you want the current Spooder-Man screenshots and rejection meme as references or placeholders while your coding agent builds the page.
  3. 03 Publish one generous scaffold. Keep the main portfolio credible, add one opt-in playful artifact, and share enough assets and credits that the next person can adapt the idea.
